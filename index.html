Distributed Version Control Systems

Typical SVN workflow
  1. checkout
  2. do some work...
  3. commit
  4. if somebody commited in the mean time - update, then go to 3.
Typical Git workflow
  1. clone
  2. do some work...
  3. add to index
  4. commit
  5. push
  6. if somebody commited in the mean time - pull, then go to 5.
